# Resume Information Extractor Specification

## Summary
Build a production-ready API that accepts PDF resumes, extracts candidate information, and returns structured JSON.

## Functional Requirements
- Accept upload of PDF files only.
- Extract candidate name, email, phone, skills, education, experience, projects, and certifications.
- Return structured data through a REST API.
- Support local development and container-based deployment.

## Non-Functional Requirements
- Maintain high code quality through linting, typing, and tests.
- Protect against common security issues and secret leakage.
- Provide clear developer documentation and automation.
