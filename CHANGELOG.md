# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-28

### Added
- Initial project structure
- FastAPI backend for resume information extraction
- React frontend for resume upload and visualization
- PDF resume upload functionality
- Automatic extraction of:
  - Personal Information
  - Skills
  - Education
  - Experience
  - Projects
  - Certifications
- REST API endpoints with Swagger documentation
- CPU-first offline processing support
- Error handling and input validation

### Improved
- Faster resume parsing
- Clean JSON response format
- User-friendly interface

### Fixed
- API endpoint issues
- File upload validation
- Minor UI and backend bugs
