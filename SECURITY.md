# Security Policy

## Supported Versions

The current release line is supported for security fixes.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by opening a security issue or contacting the maintainer directly. Do not disclose details publicly until a fix is available.

## Security Practices

- Keep dependencies updated.
- Scan for secrets and vulnerable packages in CI.
- Use least-privilege deployment settings.
- Review uploaded files and limit accepted file types.
