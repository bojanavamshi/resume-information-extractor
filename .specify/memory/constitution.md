# Project Constitution

## Purpose
This project provides a secure and maintainable resume parsing service for extracting structured candidate information from PDF files.

## Principles
- Security first: validate inputs, scan secrets, and keep dependencies updated.
- Quality first: use automated tests, linting, type checks, and coverage thresholds.
- Maintainability: keep code modular, documented, and easy to evolve.
- Reliability: containerized deployment and CI checks should be standard.
